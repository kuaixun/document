# 顶踩分享

#### **实现说明**  

顶踩分享接口*upDownOper*是提供给客户端用来累计顶、踩、分享数量的接口，其中顶和踩由客户端点击时触发，分享由客户端根据社交渠道回调参数判断分享成功后触发。

#### **请求说明**

* 请求方法 *upDownOper*

* 请求参数

| 字段 | 说明 | 类型 | 必须 |
| -- | -- | -- | -- | -- | -- |
| newsId | 新闻ID | long | 是 |
| type |操作类型，1顶，2踩，3分享 | int | 是 |
* 返回参数

* 响应示例

```
{
    "res": {
        "reCode": "1", 
        "resMessage": "Operation is successful"
    }
}
```
